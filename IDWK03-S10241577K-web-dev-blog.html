More About CSS

What I have learnt?

  • I learnt how i can make a form including different input tags and to check for validation.
  • I learnt about the different specificity such as universal, type, class and id. Using these it can help me better style my site.
  • There are also various positioning i can use such as relative, absolute, relative-absolute, fixed and Z-index. Using these different positions can help me tweak the elements on the page.
  • There is many base colours in HTML however i learnt that we can use hexadecimal mode to choose other more specific colours. It is also possible to change the text colours when the page is of different sizes using rgb.
  • The frog game has helped me better understand the use of flex-container with the use of the various codes i can use to move the position.
  • I learnt the importance of wireframes. Having low, medium and high fidelity frames can be really useful in helping me in the future to better plan out my website or app appearance.
