LOOPS

  • start from index 0
  • structure: for(initialization; condition; iteration) { // loop body }
  • Eumarative For Loops: Used for iterating over the properties of an object or the elements of an array.
  • 'for...in' iterates over keys and 'for...of' iterates over values
  • eg: for (var key in object) { console.log(key, object[key]); }
  • While loops: Structure: while (condition) { // loop body }
  • Basic While loops: Used when the number of iterations is not known beforehand. var i = 0; while (i < 5) { console.log(i); i++; }
  • Do-While loop: Guarantees the loop body is executed at least once. var i = 0; do {console.log(i); i++; while (i < 5);
  • Set: Stores unique values, removing duplicates automatically. Created using new Set() constructor. Access items using methods like add and has
  • Map: Stores key-value pairs and remembers the order of insertion. Created using new Map() constructor. Access items using methods like set, get, and has.
  • While loop: Use when the number of iterations is unknown or condition-driven. Guarantees at least one execution of the loop body.
  • For Loop: Use when a specific number of iterations is known. Compact syntax, but less explicit than a while loop in certain scenarios.
  • Infinite Loops: Be cautious to avoid unintentional infinite loops. Ensure the loop has a clear termination condition.
  • choosing Containers: Arrays are suitable for maintaining order and sequential access. Objects and Enumerative For Loops are useful for key-value pairs and non-sequential access.
