    • Week 5

    LinkedIn Learning

    What I have learnt?

    • LOOPS:
    • start from index 0
    • structure: for(initialization; condition; iteration) { // loop body }
    • Eumarative For Loops: Used for iterating over the properties of an object or the elements of an array.
    • 'for...in' iterates over keys and 'for...of' iterates over values
    • eg: for (var key in object) { console.log(key, object[key]); }
    • While loops: Structure: while (condition) { // loop body }
    • Basic While loops: Used when the number of iterations is not known beforehand. var i = 0; while (i < 5) { console.log(i); i++; }
    • Do-While loop: Guarantees the loop body is executed at least once. var i = 0; do { console.log(i); i++; } while (i < 5);

    Week 6

    What I have learnt?

    • Logical Operators:
      • Logical AND (&&):
      • True if both expressions are true.
      • Example: (2 < 5) && (3 >= 2) returns true.
      • Logical OR (||):
      • True if at least one expression is true.
      • Example: (2 < 5) || (3 >= 2) returns true.
      • Logical NOT (!):
      • Inverts true or false values.
      • Example: !true returns false.
    • Comparison and Casting:
      • Comparison Operators:
      • Use == for value equality.
      • Use === for type and value equality.
      • Casting Best Practice:
      • Use == for value equality.
      • Use === for type and value equality.
    • Functions:
      • Functions Overview:
      • Reusable code blocks.
      • Eliminate redundant code.
      • Declaring Simple Function:
      • Syntax: function functionName() { // code }.
      • Example: function sayHello() { document.write('Hello'); }.
      • Function with Parameters:
      • Syntax: function functionName(param1, param2) { // code }.
      • Example: function getArea(width, height) { return width * height; }.
      • Calling Functions:
      • Use function name with parentheses.
      • Example: sayHello();, getArea(3, 5);.
